URLe

Ścieżki URL zdefiniowane w aplikacji DrinkApp.

Główne URL

''

URL strony głównej, obsługiwany przez widok home.

Dodawanie napoju

'add/'

URL formularza dodawania nowego napoju, obsługiwany przez widok add_drink. Wymaga zalogowania.

Rejestracja

'register/'

URL formularza rejestracji, obsługiwany przez widok register.

Logowanie

'login/'

URL formularza logowania, obsługiwany przez widok login_view.

Wylogowanie

'logout/'

URL do wylogowania, obsługiwany przez widok logout_view.

Rekomendacja napojów

'recommend/'

URL systemu rekomendacji napojów, obsługiwany przez widok recommend_drink.

Widok napoju

'drink/<int:drink_id>'

URL szczegółów napoju, obsługiwany przez widok drink_view. Parametr drink_id określa ID napoju.

Polubienie napoju

'like_drink/<int:drink_id>/'

URL do polubienia napoju, obsługiwany przez widok like_drink. Parametr drink_id określa ID napoju. Wymaga zalogowania.

Negatywna ocena napoju

'dislike_drink/<int:drink_id>/'

URL do negatywnej oceny napoju, obsługiwany przez widok dislike_drink. Parametr drink_id określa ID napoju. Wymaga zalogowania.

Edycja napoju

'drink/<int:drink_id>/edit/'

URL do edycji napoju, obsługiwany przez widok update_drink. Parametr drink_id określa ID napoju. Wymaga zalogowania.

Usunięcie napoju

'drink/<int:drink_id>/delete/'

URL do usunięcia napoju, obsługiwany przez widok delete_drink. Parametr drink_id określa ID napoju. Wymaga zalogowania.
